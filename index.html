Top 10 web languages you should know!

JavaScript
The first JavaScript engine was created on December 4, 1995 by Brenden Eich at NetScape.
See Wiki.
Ruby
Ruby is a dynamic, reflective, object-oriented, general-purpose programming language. It was designed in the mid 1990s by Yukihiro Matsumodo in Japan.
See Wiki.
Java
Java was originally developed by James Gosling at Sun Microsystems, which has since been acquired by Oracle and released in 1995 as a core component of Sun Microsystems' Java Platform.
See Wiki.
CSS
The CSS Working Group tackled issues that had been addressed with CSS Level 1 resulting in creation of Level 2 on November 4, 1997. It was published as a W3C recommendation on May 12, 1998.
See Wiki.
Python
Python was conceived in the late 1980s and its implementation began in December 1989 by Guido Van Rossum at CWI.
See Wiki.
C#
James Gosling, who created the Java programming language in 1994, and Bill Joy, a cofounder of Sun Microsystems, the originator of Java, called C# an imitation of Java.
See Wiki.
C++
Before the initial standardization in 1998, C++ was developed by Bjarne Stroustrup at Bell Labs.
See Wiki.
Go
Go was released by Francis McCabe on November 10, 2009. He had spent 10 years developing this language.
See Wiki.
PHP
PHP is known today as the successor to PHP/FI. Created in 1994 in Rasmus Lerdorf.
See Wiki.
Perl
Perl was originally developed by Larry Wall in 1987 as a general purpose UNIX scripting language to make report processing easier.
See Wiki.
